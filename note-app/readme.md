# Simple Note App using node js.

### Npm packages used in this app.

* yargs --> for command line arguments.

### Usage

#### node app.js --options--

* For help use
  ```bash
  node app.js --help
  ```

#### Options

* add a note


```bash
  node app.js add
```

* list all notes


```bash
  node app.js list
```

* read a note


```bash
  node app.js read
```

* remove a note


```bash
  node app.js remove
```

#### For individual command options

```bash
node app.js add --help
```
